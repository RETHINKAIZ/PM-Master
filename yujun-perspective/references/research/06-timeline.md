# 俞军人物时间线研究

| 时间 | 事件 | 对思维框架的影响 |
|---|---|---|
| 2000 年前后 | 以网名“搜索引擎 9238”写求职信，自学搜索知识 | 形成对信息获取成本、搜索体验和用户行为的长期兴趣 |
| 2001-09 | 百度搜索首页上线后参与百度早期产品工作 | 在高成长搜索产品中获得高密度实践 |
| 2003 | 百度贴吧等产品阶段出现 | 从搜索需求扩展到用户内容和社区行为 |
| 2005 | 百度知道等产品阶段出现；百度成为访问量最大的中文网站 | 强化“用户需求、内容供给、产品体验”之间的系统关系 |
| 2005 前后 | “俞军产品经理十二条”成型 | 早期用户体验与用户视角方法论沉淀 |
| 2009-06 | 离开百度，留下人才选用育留讲座和感谢信 | 从产品实践转向人才、组织和方法论沉淀 |
| 2016 | 加入滴滴 | 进入海量实时多边交易场景，推动从用户模型到交易模型的升级 |
| 2017 | 在知乎/三节课等场景分享产品经理方法论；接受深度访谈 | 方法论开始外部化和系统化 |
| 2019-06 | 因家庭原因离开滴滴，仍表示会继续参与和共同成长 | 方法论阶段性沉淀，进入书稿整理 |
| 2020-01 | 《俞军产品方法论》出版 | 公开呈现用户模型、交易模型、理性决策和产品经理成长体系 |
